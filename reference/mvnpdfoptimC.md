# mvnpdfoptimC

mvnpdfoptimC

## Usage

``` r
mvnpdfoptimC(x, mean, varcovM, Log = TRUE)
```

## Arguments

- x:

  matrix x

- mean:

  column vector of x's mean

- varcovM:

  matrix of x's variance-covariance

- Log:

  boolean. If true, return log values of x's multivariate normal
  distribution
